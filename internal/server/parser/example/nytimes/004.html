馬斯克為何對「X」情有獨鍾

STELLA BUGBEE
2023年7月27日
這個字母有它明顯的魅力。X形狀靈活，是少數幾個無論在垂直或水平等分都能保持對稱的大寫字母之一（其他三個是H、O和I），無論怎麼翻轉，它都很容易辨認。它預示著結束。地圖上的一個標記。一部電影太過分了，沒法給它評級，就用X來代表。它屬於革命者（馬爾科姆·X）和龐克（樂隊X）。它代表親吻，在數學中代表一個未知量。XXX是色情——互聯網上第一種真正的貨幣。X可以是《X檔案》(The X-Files)中隱藏的陰謀，也可以是搖頭丸(Ecstasy)的簡稱，在90年代的狂歡派對上，這種毒品很受歡迎。
週一，Twitter公布了它的新名字和新標識。自伊隆·馬斯克去年10月收購Twitter以來，它的每一步發展都充滿了混亂的能量，新品牌的發布也不例外。X（它之前曾被一些人半帶親昵地稱為「鳥應用」）的用戶在適應新環境的過程中，有很多問題需要思考。比如：如果它不再被稱為Twitter，那麼帖子還能叫推文嗎？X飛過矽谷會發出什麼樣的聲音？嗖嗖聲？馬斯克為什麼選擇它？
可能與他的年齡有關。20世紀90年代，在道格拉斯·庫普蘭1991年的小說《X世代——加速文化的故事》(Generation X: Tales for a Accelerated Culture)風靡一時後，「X」佔據了崇高的地位。
「我們被稱為X世代時才20多歲，」50歲的安東尼·斯佩杜蒂說道，他是品牌工作室——神話的創始人。「所以，也許X對我們來說感覺不錯，因為它滲入了我們的大腦。」現年52歲的馬斯克正好屬於這個群體。
廣告
在20世紀90年代，X用於營銷確實顯得有點酷。它體現了對權威的拒絕——你可以想像巴特·辛普森用記號筆在卧室的牆上寫X的樣子——同時也被大眾消費主義所同化。X是非商標產品的象徵，所以X可以是叛逆的，也可以是批量生產的。
大企業也加入了「X」的行列，試圖將這十年的另類能量裝入瓶中。極限運動會(X Games)始於1995年，X是「極限」(extreme)一詞的縮寫，指的是單板滑雪等運動，直到1998年才被允許進入奧運會。2001年，Xbox影片遊戲機問世。
音樂家和潮人在創建公司時大量使用這個字母。1991年，艾利·博納茲和亞當·席爾瓦曼在洛杉磯開了服裝店XLARGE（它是「小獸孩」的最愛）。金·戈登和黛西·馮·福士於1994年創立了她們的服裝品牌X-Girl。
但是X也有它的局限性。
「作為一種標識形象，它沒有意義，」64歲的麥可·洛克在談到新標識時說。他是總部位於紐約和北京的品牌諮詢公司2x4的合伙人。「它的意義在於否定或取消。劃掉。」
不論如何，X世代是以字母作為世代名稱的第一代人，即使他們不喜歡這個綽號，他們也為自己的無法定義而感到自豪。總之，這個名字被保留了下來。而且，它似乎真的在馬斯克的心裡留下了最長久、最強烈的印象。（馬斯克沒有回應置評請求。）
廣告
的確，馬斯克似乎對字母X情有獨鍾。他給自己的火箭公司命名為SpaceX，給一款特斯拉車型命名為Model X，給自己的一個孩子起名叫X Æ a-12，簡稱X。馬斯克的第二家公司X.com成立於1999年，那是「X世代的風頭還在繼續」的年份，2001年，該公司與另一家公司合併，成為PayPal。
作為一個單獨的字母，X在打造品牌方面幾乎沒有競爭對手，直到1998年，蘋果公司在iMac的命名中使用了小寫i。與X的暴力斜線相比，i顯得樂觀、自我指涉，而且具有歡快的擬人化風格——一個小圓點，就像一個腦袋，棲息在一個小巧挺拔的身體上，準備接管世界。你可能會說，這是一個千禧年的字母，用來取代自我否定的X和它無人管束的能量。
馬斯克用X來標記他接觸到的一切，似乎是在乞求世界記住他開始建立自己帝國的黃金歲月（這不是典型的X一代本能）。
洛克說，在某種程度上，「這是一種別無選擇的選擇，先不提供背後的敘事，日後再說。」他認為，該公司在網站上放置什麼樣的標誌並不是重點。「反常似乎是它的核心。」
洛克把它比作「特斯拉發布會上的摩托車皮夾克」，更像是Xbox或 X Games的品牌塑造，並沒有上世紀90年代那種真正的反主流文化用途。
「我可以想像，它符合伊隆·馬斯克另類、前衛、神秘或龐克的形象，但它給人的感覺是科技小子的尷尬感，」他說。
廣告
最後，如果馬斯克成功地把X打造成他想要的「無所不能的公司」，那麼這個標誌也就無所謂了。這也許是最符合X世代的結果。
